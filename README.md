# 🎮 Quiz Game in C

A simple and fun Quiz Game built using the C Programming Language. It consists of two engaging rounds to test your general knowledge!

## 🧩 Game Structure

### 1. Basic Round
- Total Questions: **3**
- Level : Easy

### 2. Challenge Round
- Total Questions: **10**
- Level : Hard

## 🛠️ Tech Stack
- Language: **C**
